# Game Mode Rules

> Traceability: Wikipedia, Steam store page, patch notes (v1.3.5 map select).

## Free-for-All

- **Players**: 1–6 (humans + bots)
- **Win condition**: most kills within round timer (default 120s) or first to kill limit
- **Respawn**: 2s delay, invulnerable 1s
- **Friendly fire**: off by default (configurable)
- **Scoring**: 1 point per kill; environmental deaths don't score

## Teams (Spicy vs Chill)

- **Teams**: 2 (Spicy = red, Chill = blue)
- **Win condition**: team with most kills at timer
- **Friendly fire**: off within team
- **Assignment**: auto-balance or manual in lobby

## Golden Boomerang

- **Special weapon**: one golden boomerang spawns mid-round
- **Holder effects**: slower movement (0.7×), heavier throw (1.3× damage radius)
- **Scoring**: kills with golden boomerang worth 3 points
- **Drop**: holder death drops golden boomerang for pickup

## Hide & Seek

- **Phases**: Hide (30s) → Seek (90s)
- **Hiders**: start with disguise power-up; no boomerang
- **Seekers**: armed; must identify and kill hiders
- **Win**: seekers kill all hiders (seeker win) or timer expires with hiders alive (hider win)
- **Disguise props**: arena-specific prop set required

## Match Flow (all modes)

1. Countdown 3-2-1
2. Round active
3. Round end → scoreboard
4. XP awarded → unlock check
5. Map vote / next round

## Rules Editor (meta)

Configurable per lobby:

- Round timer (60–300s)
- Kill limit (0 = time only)
- Power-up pool (enable/disable types)
- Power-up spawn rate
- Shield for last-place player
- Friendly fire toggle
